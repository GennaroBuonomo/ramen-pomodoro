# 🍜 Ramen Pomodoro

Un timer Pomodoro rivisitato, interattivo e visivamente rilassante, sviluppato con **React** e **TypeScript**.

Invece del classico e ansiogeno conto alla rovescia numerico, questo progetto utilizza un approccio "Zen": la concentrazione viene premiata visivamente. Durante i 25 minuti di focus, una ciotola di ramen vuota si riempie progressivamente di ingredienti (noodles, uovo, chashu, cipollotto). Se il timer viene interrotto, i progressi svaniscono.

## ✨ Funzionalità

- **Focus Visivo:** Animazioni CSS personalizzate legate all'avanzamento del tempo.
- **Cicli Pomodoro:** Gestione accurata delle fasi di lavoro (25 min), pausa breve (5 min) e pausa lunga (15 min).
- **Architettura Pulita:** Netta separazione tra la logica di calcolo del tempo (Custom Hooks) e il rendering grafico dell'interfaccia.
- **Tipizzazione Rigorosa:** Uso di TypeScript per garantire stabilità negli stati del timer e delle animazioni.

## 🛠️ Tech Stack

- **Core:** React 18
- **Linguaggio:** TypeScript
- **Build Tool:** Vite
- **Styling & Animazioni:** CSS3 (Keyframes, Flexbox, SVG manipulation)
